__Требования:__ \
Проверка контента (отображение текста, логотипа, контактных данных, видео, ссылок)\
Проверка функций (кнопки, переход на другие разделы)\
Проверка UI (адаптивность для ПК или мобильных устройств)\
__Сценарии:__ \
Пользователь знакомиться с материалом и успешно оставляет заявку на обучение\
Пользователь знакомиться с материалом, но после отправки заявки происходит ошибка\

На мобильных устройствах форма выглядит некорректно\
