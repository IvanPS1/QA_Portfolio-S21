**Тестовый сценарий по процессу "Как поступить в Школу 21":**

__Проверка регистрации в "Школа 21"__

1) Открыть страницу https://21-school.ru/admission 
2) Нажать на любую из кнопок "Поступить в школу"/"Поступить" 
Ожидаемый результат: Открывается форма регистрации 
3) В выпадающем списке "Выбери кампус" выбрать город 
4) В выпадающем списке "Выбери программу" выбрать программу обучения 
5) В поле "Электронная почта" ввести валидный mail-адрес 
6) Установить флажок "Подтверждаю свое согласие на обработку персональных данных. Политика обработки персональных данных" 
7) Нажать кнопку "Поступить" 

**Ожидаемы результат:** Пользователя переносит на сайт апликанта (https://applicant.21-school.ru/) для дальнейшей регистрации. На почту приходит письмо с подтверждением
