## Task 3. Решение

### Баг-репорт 1:

**Заголовок:** Цены товаров на странице /inventory.html изменяются после обновления страницы для пользователя visual_user

**Продукт:** https://www.saucedemo.com/

**Компонент продукта:** inventory.html

**Версия продукта:** Sauce Labs 2025

**Критичность:** Critical

**Приоритет:** High

**Статус:** New

**Автор:** karoncre

**Назначен на:** Команда разработчиков

**Окружение:** macOS Sequoia 15.5, Google Chrome Версия: 136.0.7103.114

**Шаги воспроизведения:**
1. Открыть сайт https://www.saucedemo.com/ 
2. Ввести логин visual_user в поле "Username"
3. Ввести пароль secret_sauce в поле "Password"
4. Нажать кнопку "Login"
5. Зафиксировать текущие цены товаров
6. Обновить страницу
7. Сравнить цены до перезагрузки страницы и после

**Фактический результат:** После перезагрузки страницы с товарами, цены карточек остаются прежними

**Ожидаемый результат:** После перезагрузки страницы с товарами, цены карточек меняются
