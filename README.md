## Структура и эволюция моих навыков

Мое обучение было построено по принципу «от простого к сложному»:

### Фундамент & Тест-артефакты
*   **QA1: Введение в тестирование** - основы профессии, философия тестирования
*   **QA2: Тестовые артефакты** - создание тест-кейсов, чек-листов, баг-репортов и тест-планов
*   **QA5: Дефекты** - углубленное изучение жизненного цикла бага, классификация и локализация дефектов

### Тест-дизайн & Анализ
*   **QA3: Тест-дизайн и тест-анализ** - практическое применение ключевых техник:
    *   Классы эквивалентности, анализ граничных значений
    *   Составление тестовых туров и расчет тестового покрытия
*   **QA4: Виды и уровни тестирования** - тестирование документации, методики Black-box и Grey-box

### Специализация: Веб, Мобильные и Backend
*   **QA6: Тестирование Web** - работа с Chrome DevTools, кроссбраузерное тестирование, валидация HTML/CSS, UI/UX тестирование по макетам Figma
*   **QA8: Мобильное тестирование** - работа с эмуляторами (Android Studio, Xcode), логирование (Logcat), анализ трафика (Charles Proxy)
*   **QA7: Backend-тестирование** - тестирование REST API (Postman, Swagger), работа с JSON, основы SQL и различия СУБД

### Продвинутые темы
*   **QA9: Основы безопасности в Web** - принципы InfoSec (Confidentiality, Integrity, Availability), основы криптографии, анализ уязвимостей.

### Программирование & Автоматизация (В процессе изучения)
*   **Java_Bootcamp (Project 01, 02)** - основы Java, ООП, функциональная парадигма, многопоточность, итераторы
*   **Java_Bootcamp (Project 03)** - основы веб-приложений на Spring (MVC)
*   **SQL Bootcamp (Day 00, 01)** - написание SQL-запросов (SELECT, JOIN, UNION, подзапросы), работа с реляционной моделью данных
