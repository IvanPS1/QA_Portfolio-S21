## Task 5. Grey-box

У тебя как тестировщика данного приложения есть доступ к базам данных в которых хранится информация о магазинах.

Основываясь на данных, указанных выше, напиши чек-лист проверок по методике  Grey-box.

## Task 5. Решение: 

#### 1. Безопасность
* Проверка SQL инъекции в поле ввода названия магазирна
* Проверка XSS инъекции в поле ввода названия магазина
* Проверка доступа к БД без авторизации 
* Провекра обработки спецсиволов
#### 2. Данные магазинов
* Проверка соответсвия названия магазина с результатми в БД
* Проверка учета закртых магазинов при поиске и как это регулируется в БД
* Проверка наличия дубликатов магазинов
#### 3. Геоданные и маршурт
* Проверка формул постороения ближайшего маршрута в БД
* Проверка кэширования маршурата (для продолжения маршрута в случае перезагрузки приложения)
* Проверка геоданных в БД
#### 4. История и производительность
* Проверка истории поиска (где хранится?)
* Проверка времени ответа от БД
* Проверка нагрузки на БД при одновременных запросах
