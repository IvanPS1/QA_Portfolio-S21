2. Опиши назначение каждой вкладки инструмента с точки зрения полезности тестировщику.

## Task 2. Решение

### 1.

 * **Chrome DevTools** - это набор инструментов для разработчиков и тестировщиков, встроенный в браузер Google Chrome, который позволяет анализировать, отлаживать и оптимизировать веб-сайты и приложения. Он предоставляет функциональность для изучения HTML, CSS и JavaScript кода страницы, а также для мониторинга сетевой активности и производительности

 * Чтобы открыть **Chrome DevTools**, можно воспользоваться одним из следующих способов: нажать клавишу F12 (для Windows), использовать сочетание клавиш Ctrl + Shift + I (или Cmd + Option + I для macOS), или через меню браузера, выбрав "Дополнительные инструменты" -> "Инструменты разработчика"

### 2.

| Вкладка | Назначение | Пример использования для тестировщика |
| - | - | - |
| **Elements** | Анализ и изменение DOM/CSS | Проверить, почему кнопка не отображается  |
| **Console** | Логирование ошибок, выполнение JS | Найти ошибки JavaScript |
| **Sources** | Отладка JavaScript | Найти, где "сломалась" анимация на сайте |
| **Network** | Мониторинг запросов | Анализировать время загрузки ресурсов |
| **Performance** | Анализ скорости работы | Проверить, не тормозит ли сайт при скролле |
| **Memory** | Поиск утечек памяти | Убедиться, что сайт не "ест" память при долгой работе |
| **Application** | Работа с хранилищами  | Посмотреть, сохранился ли логин в куках после перезагрузки |
| **Security** | Проверка HTTPS | Проверить, есть ли защита на сайте |
| **Lighthouse** | Автоматический аудит | Узнать, как выглядит сайт на телефоне |
